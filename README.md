zashboard

A modern, feature-rich dashboard for Clash API (supporting both mihomo and sing-boxcores). Built with Vue 3, TypeScript, and Tailwind CSS, offering a beautiful Apple-inspired design with extensive customization options.

 Features

- Comprehensive Proxy Management**: View and manage proxy groups, nodes, and providers with real-time latency testing
- Real-time Connection Monitoring**: Track all active connections with detailed information and filtering
- Rule Management**: View, search, and manage routing rules and rule providers
- DNS Query Visualization**: Inspect DNS resolution details and history
- Log Viewer**: Real-time log streaming with filtering and search capabilities
- PWA Support**: Install as a Progressive Web App for native-like mobile experience
- Multi-language Support**: Chinese (Simplified & Traditional), English, Russian, and more
- Theme Customization**: Multiple built-in themes including Apple-style dark/light modes, plus custom theme support
- Responsive Design**: Optimized for both desktop and mobile devices
- Extensive Settings**: Configure UI preferences, backend connections, and proxy behavior
- Auto-Upgrade**: Check for core updates and upgrade directly from the dashboard

Requirements

- Chrome 111+ (March 2023)
- Firefox 128+ (July 2024)
- Safari 16.4+ (March 2023)
- Not supported on iOS 16.4 jailbroken version

 Manual Deployment
1. Download the latest release from [GitHub Releases](https://github.com/Unwilledflow/zashboard-ios-style-adn-m3e/releases/tag/)
2. Extract the archive
3. Serve the files using any static web server (Nginx, Apache, etc.)
